# Koereakademiet Block Theme

This is a Gutenberg block theme starter for the homepage conversion.

Contents:
- `style.css`: theme header and styling.
- `theme.json`: global block theme settings.
- `templates/front-page.html`: front page template using page content.
- `parts/header.html` and `parts/footer.html`: shared template parts.
- `patterns/homepage.php`: insertable homepage pattern built with native blocks.

Recommended usage:
1. Zip the `koereakademiet-block-theme` folder and install it as a WordPress theme.
2. Activate the theme.
3. Create a page named `Forside`.
4. In the page editor, insert the pattern `Homepage`.
5. Set that page as the WordPress front page under `Settings -> Reading`.
6. Configure navigation links in WordPress.
